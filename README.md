**********************************************
A Python programme to print Terminal Commands.
**********************************************

The idea for this small project came from teaching a friend the basics of Linux & Python.
He had never used a command line before so I thought writing a small Python project that 
prints helpful Terminal Commands, he could also see the process of the Python Code.

I've also decided to run the entire project via GitHub so that he can gain experience with 
how the platform works. 
I will get him to fork the repo and actively edit the code as a way of learning.

Initially, I will concentrate on the following sections:

1) File Commands
2) SSH
3) Installation
4) Network
5) System Information
6) Permissions
7) Shortcuts

Later in the project, I will include some Process & VIM commands.
